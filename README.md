### Play with Spring

通过几种不同的技术选型来实现相同的业务功能，在这个过程中，体会不同的技术，比较不同的技术。



业务也是很简单的CURD，模拟一个书店的场景。需要对书籍进行增加、删除、修改和查看。

期间也会用到一些其他的小技术，通过完成这些小demo，使自己对框架和业务的理解更加深刻。

同时，参考其他作者的优秀文档和设计规范，深刻体会面向对象、解耦等思想。
